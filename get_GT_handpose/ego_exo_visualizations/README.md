# Ego/Exo ChArUco and Hand Triangulation Outputs

Generated local artifacts for the stationary exo-camera ChArUco experiment.

These files include:

- ChArUco extrinsic calibration reports and overlays.
- Marker triangulation sanity checks.
- Stereo hand-labeling HTML tools.
- Left-hand 21-keypoint triangulation outputs.
- Bone-length comparisons against the measured personal hand model.
- World-frame board/camera/hand visualizations.

Raw camera videos are intentionally not included here.
